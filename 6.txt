Типы данных в EDA. Предобработка данных. Инструменты визуализации EDA. Визуализация зависимостей признаков в наборе данных.

Типы данных в EDA.
- числовые: делятся на непрерывные и дискретные(ограниченное количество вариантов) 
- категориальные: делятся на порядковые (имеют иерархию) и номинальные (не имеют иерархию)
Предобработка данных. Удалить признаки, не имеющие отношения к задаче (например, ID) Категориальные данные можно преобразовать в числовые путем: one-hot кодирования с помощью map, т.е. каждому значению присваивается свое число и меняется на него. Заполнение пропусков в числовых признаках можно сделать с помощью заполнения средним, медианой, модой или с помощью группировки с другим признаком. Стандартизация данных с помощью StandartScaler/MinMaxScaler: 
Z=(x_i-x ̅)/s, где S - стандартное отклонение 
Удаление дубликатов с помощью df.drop_duplicates(). Преобразование типов данных (в инт, флоат, datetime и тд).
Инструменты визуализации EDA: matplotlib, seaborn.
Визуализация зависимостей признаков в наборе данных. 
	Гистограмма - графическое представление распределения данных по различным интервалам. 
	Диаграмма рассеяния (scatter plot) - график, в котором каждая точка представляет собой отдельное наблюдение и показывает взаимосвязь м/у двумя переменными. 
	Тепловая карта (heat map) - графическое представление матрицы данных, где цветовая шкала показывает степень взаимосвязи между переменными. 
	Круговая диаграмма (pie chart) - используется для номинальных и порядковых переменных с числом категорий до 5. 
	Ящик с усами (box plot) - отображает распределение количественных данных, показывая их основные статистические характеристики. Он особенно полезен для выявления выбросов, оценки разброса и сравнения распределений между группами.
